Configure DNS for GitHub Pages

These are the DNS records you need to create on your DNS host in order for GitHub pages to work.

A Records:
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

CNAME record:
Point "www" to your GitHub Pages repository.
For instance www.stegratech.com points to "stegratech.github.io"
